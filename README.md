# Rating Scale Analysis (RSA) - Passion Dataset

## Project Overview
This project analyzes a rating scale dataset called "Passion" using R. The analysis includes statistical methods for understanding rating patterns and survey responses.

## Files in This Repository
 `Passion.xlsx` Raw rating scale dataset 
 `RSA Project.Rmd` Complete R Markdown script (reproducible analysis) 
 `RSA-Project.html` Full HTML report (view in browser) 

## What This Analysis Includes
- Data cleaning and preparation
- Descriptive statistics
- Rating scale analysis (RSA)

## Tools Used
- R Programming Language
- R Markdown for reproducible reporting
- Key packages: tidyverse, dplyr, ggplot2, TAM, psych, readexcel, WrightMap.

## How to View the Report
Click on `RSA-Project.html` above, then click "View raw" or "Download" to open in your browser.

## How to Reproduce This Analysis
1. Download `Passion.xlsx` and `RSA Project.Rmd`
2. Open the .Rmd file in RStudio
3. Install required packages
4. Knit the file to regenerate the report
